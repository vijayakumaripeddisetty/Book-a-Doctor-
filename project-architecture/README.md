Book a Doctor
Project Overview
Book a Doctor is a full-stack healthcare appointment booking system developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js). The application enables patients to search for doctors, schedule appointments, upload medical documents, and manage their healthcare records. Doctors can manage appointments and patient requests, while administrators monitor and manage the platform through role-based access control.

⦁	Technical Architecture
Architecture
The application follows a three-tier client-server architecture:
⦁	Frontend: React.js, Bootstrap, Material UI, Axios
⦁	Backend: Node.js, Express.js, JWT, bcrypt
⦁	Database: MongoDB, Mongoose
